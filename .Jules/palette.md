## 2024-03-23 - Добавлен aria-current="page" к активным ссылкам навигации
**Learning:** В приложении используется множество компонентов навигации (MobileBottomNav, LiteMobileBottomNav, DesktopSidebar, AppHeader, AppShell), которые используют функцию `isActive(path)` для визуального выделения активных маршрутов. Однако ни в одном из них не использовался атрибут `aria-current="page"`, который критически важен для скринридеров, чтобы пользователи понимали, на какой странице они находятся.
**Action:** При добавлении новых компонентов навигации или ссылок, которые визуально отображают текущее состояние страницы, всегда следует добавлять `aria-current={isActive(path) ? 'page' : undefined}` для обеспечения доступности.
