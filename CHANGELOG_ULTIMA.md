# Обновление: Режим Ultima и поддержка анимаций

## Что было сделано

В ходе этого обновления была добавлена возможность полной кастомизации дизайна страницы режима **Ultima** прямо из панели администратора, а также поддержка динамических фоновых анимаций.

### 1. Переработка стилей в приложении (CSS & Tailwind)
- Жёстко заданные цвета в Tailwind (например, произвольные hex-коды) были удалены из компонентов `UltimaDashboard.tsx` и `UltimaBottomNav.tsx`.
- Вместо них теперь используются динамические CSS переменные (`var(--ultima-accent)`, `var(--ultima-bg)`, `var(--ultima-radius)` и т.д.).
- Это позволяет интерфейсу моментально реагировать на изменение настроек темы без перезагрузки и изменения исходного кода.

### 2. Адаптивные Layout'ы (Контейнеры)
- Добавлен новый класс `.ultima-layout-full` в `globals.css`.
- Динамически применяя этот класс, приложение может переключаться между **центрированным (Widget) режимом** (с отступами) и **полноэкранным (Full-screen) режимом**, который автоматически занимает все свободное место как на ПК, так и на мобильных устройствах (используя `100dvh`).

### 3. Интеграция фоновых анимаций
- Добавлен новый компонент `UltimaBackgroundRenderer.tsx`.
- Он переиспользует уже существующие эффекты (Аврора, Искры и т.д.) из `backgroundRegistry`, но рендерит их в изолированном слое `z-0` внутри `.ultima-shell`, чтобы они не перекрывали кликабельные элементы дашборда.
- В хук темы `useUltimaTheme.ts` добавлен парсинг `AnimationConfig`, который позволяет загружать настройки размытия (blur), прозрачности и типа выбранной анимации.

### 4. Панель управления (Admin Panel)
- Создан новый роут `/admin/ultima-settings/theme`.
- Разработан компонент `AdminUltimaThemeEditor.tsx`, который предоставляет удобный графический интерфейс для настройки:
  - Основного цвета (Accent).
  - Цвета фона (Background).
  - Скругления углов (Border Radius).
  - Режима отображения (Полноэкранный / По центру).
  - Настроек анимации (Вкл/Выкл, выбор эффекта, ползунки прозрачности и размытия).
- Данные сохраняются в виде единого JSON объекта в настройку `ULTIMA_THEME_JSON` через API `adminSettingsApi`.

### 5. Рефакторинг и исправления типов (TypeScript)
- Изолирован компонент `SettingField.tsx` для переиспользования полей ввода в панели администратора.
- Исправлены проблемы с типами React Query.
- Все файлы прошли строгие проверки линтеров (`npm run type-check` && `npm run lint`). Моки удалены, проект готов к деплою на боевой сервер.
